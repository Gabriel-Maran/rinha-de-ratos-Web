# README

Ao clonar o repositório, não esqueça de realizar os seguintes passos:

* Rodar o back end
* Adicionar o que está dentro de `SQL.sql` diretamente no PgAdmin
* Rodar `npm install` e `npm run dev` no front end

## Stack

* Back end: Spring com Kotlin
* Front end: React + JavaScript
* DB: Postgres
